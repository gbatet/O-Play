# O-Play
## Orienteering Post Race Analysis
--------------------------------------
Python code for the O-Play program.

### O-Play Workflow:
1. Open the program
2. Load a map image (jpg, jpeg or png)
3.0. Click on 3 points and introduce their coordinates to calibrate the map, then save the calibration into a *.txt file
3.1. If a cal file already exists, upload it via "Load Cal"
4. Load GPX file
5. LET'S GO AND ANALYSE!
